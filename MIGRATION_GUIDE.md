# 댕댕로드 - LLM → 규칙기반 알고리즘 전환 안내

## 🎯 주요 변경사항

### 1. **LLM (Groq API) 완전 제거**
- ❌ Groq 의존성 제거 (API 키 삭제)
- ❌ LLM 기반 코스 생성 로직 삭제
- ✅ **규칙기반 알고리즘**으로 즉시 코스 생성

### 2. **새로운 규칙기반 코스 생성 엔진**

#### 핵심 함수들:

**`classifyPlace(place, keywords)`**
- 장소를 자동으로 카테고리 분류
- 분류: 도그런, 수영장, 카페, 식당, 숙박, 공원, 미용, 병원, 펫샵
- 우선순위 기반 정규식 매칭으로 정확한 분류

**`getCourseCombinations(activity, categorized)`**
- 활동별 코스 조합 규칙 정의:
  - **냄새 탐험**: 공원 + 카페
  - **물놀이**: 수영장 + 카페/식당
  - **에너지 발산**: 도그런/운동장 + 카페 (공원 제외)
  - **친구 만나기**: 펫카페 + 식당/공원
- 각 활동마다 최대 15가지 조합 생성

**`buildCourse(template, categorized, ...)`**
- 선택된 템플릿을 기반으로 실제 코스 구축
- 거리 검증 (10km 이내)
- 이동시간, 체류시간 자동 계산
- 추천시간대 자동 설정

**`generateCoursesByRules(places, ...)`**
- 모든 장소 데이터에서 유효한 코스 5개까지 생성
- 다양성 확보를 위해 장소 중복 사용 방지

## 📊 코스 생성 알고리즘 상세

### 단계별 처리:

```
1. 장소 데이터 수집
   ├─ 한국관광공사 API (반려동물 공식 인증)
   └─ 카카오맵 검색 (다양한 선택지)

2. 거리 필터링
   ├─ 최소거리(minKm) ≥ 장소거리
   └─ 장소거리 ≤ 최대거리(maxKm)

3. 카테고리 분류
   ├─ 정규식 패턴 매칭
   └─ 9가지 카테고리로 자동 분류

4. 활동별 조합 규칙 생성
   ├─ 활동 타입에 맞는 2개 카테고리 조합
   └─ 최대 15가지 조합 템플릿 생성

5. 코스 구축
   ├─ 각 템플릿으로 실제 코스 생성 시도
   ├─ 거리, 반복 검증
   └─ 유효한 코스 5개까지 수집

6. 반환
   ├─ 5개 코스 중 랜덤 3개 선택
   └─ 다양성 확보
```

## 🔧 활동별 코스 구성 규칙

### 냄새 탐험 (Park + Cafe)
```javascript
{
  primary: '공원' (산책로, 자연 탐험)
  secondary: '카페' (휴식)
  rule: "자연 속에서 냄새 탐험 후 카페에서 휴식"
}
```

### 물놀이 (Pool + Cafe/Restaurant)
```javascript
{
  primary: '수영장' (물놀이)
  secondary: '카페' 또는 '식당' (식사/휴식)
  rule: "물놀이 후 근처 카페/식당에서 휴식"
}
```

### 에너지 발산 (DogRun + Cafe)
```javascript
{
  primary: '도그런' (자유로운 운동)
  secondary: '카페' (휴식)
  rule: "도그런에서 충분히 에너지 발산 후 카페"
  constraint: "공원 2개 조합 금지"
}
```

### 친구 만나기 (PetCafe + Restaurant/Park)
```javascript
{
  primary: '카페' (펫카페, 사교)
  secondary: '식당' 또는 '공원' (추가 체험)
  rule: "펫카페에서 친구를 만나고 식사/산책"
}
```

## 🎮 테스트 방법

### 1. 서버 실행
```bash
node server_ruleset.js
```

### 2. API 호출 (예시)
```bash
curl -X POST http://localhost:3000/api/generate-course \
  -H "Content-Type: application/json" \
  -d '{
    "lat": 37.5665,
    "lng": 126.9780,
    "activity": "냄새 탐험",
    "duration": "1시간 거리",
    "dogName": "뽀삐",
    "dogBreed": "말티즈",
    "dogSize": "small"
  }'
```

### 3. 응답 확인
- ✅ 즉시 결과 반환 (LLM 대기 없음)
- ✅ 코스 3~5개 제공
- ✅ `"algorithm": "rule-based"` 표시

## 📱 프론트엔드 호환성

HTML/JS는 **기존과 동일**하게 작동합니다:
- 동일한 API 응답 구조
- 동일한 JSON 포맷
- 추가 수정 불필요

```javascript
// 기존 코드 그대로 사용 가능
fetch('/api/generate-course', { ... })
  .then(res => res.json())
  .then(data => {
    // data.courses 는 동일한 구조
    // UI는 변경 없음
  });
```
